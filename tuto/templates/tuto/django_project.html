Comment initier un projet avec django, conda et git

Initialisation du projet

Activer un environnement conda spécifique au projet : 

conda activate my_env

Initier un projet : 

django-admin startproject mysite

La commande ci-dessus va créer un dossier mysite à l’endroit où le terminal se trouve. Pour pouvoir accéder à ce dossier il faut utiliser la commande :

cd mysite

Une fois que vous vous êtes placés dans le dossier contenant votre projet, c’est le bon moment pour initialiser git :

git status

La commande précédente permet de vérifier que vous vous ne trouvez pas déjà dans un dossier git (un dossier parent par exemple, si c’est le cas, vous devez menez l’enquête et supprimer le dossier .git responsable)
